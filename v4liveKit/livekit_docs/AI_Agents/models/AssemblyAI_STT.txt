model

string

Required

#

The model to use for the STT.

language

string

Optional

#

Language code for the transcription. If not set, the provider default applies.

extra_kwargs

dict

Optional

#

Additional parameters to pass to the AssemblyAI Universal Streaming API, including

format_turns

,

end_of_turn_confidence_threshold

,

min_end_of_turn_silence_when_confident

,

max_turn_silence

, and

keyterms_prompt

. See the provider's

documentation

for more information.

In Node.js this parameter is called

modelOptions

.
