BarVisualizer

Note

This feature is under active development and may change based on developer feedback and real-world usage.

Visualizes audio signals from a TrackReference as bars. If the

state

prop is set, it automatically transitions between VoiceAssistant states.

Import

import

{

BarVisualizer

}

from

"@livekit/components-react"

;

Remarks

For VoiceAssistant state transitions this component requires a voice assistant agent running with livekit-agents >

= 0.9.0
